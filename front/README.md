# front

## Project setup
```
npm install
```

### Axios setup (optional)
```
Go to "src/services/Api.js" and modify baseUrl. 
By default is "http://localhost:8000/api"
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
